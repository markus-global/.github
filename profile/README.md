# Markus Global

[![Website](https://img.shields.io/badge/Web-markus.global-blue?style=flat-square)](https://www.markus.global/)
[![GitHub](https://img.shields.io/badge/GitHub-markus--global%2Fmarkus-181717?style=flat-square&logo=github)](https://github.com/markus-global/markus)

**The open-source operating system for AI workforces.**

Markus is a production-grade platform where autonomous AI agents work alongside your team — coordinating, remembering, reviewing each other's output, and delivering real results. One command to launch, fully self-hostable, no vendor lock-in.

We believe the future of work is humans **with** AI, built in the open.

### Get Started

```bash
npm install -g @markus-global/cli
markus start
```

Or try the Linux one-liner:

```bash
curl -fsSL https://markus.global/install.sh | bash
```
